Tread Lightly
Co-op horror puzzle game

By Jacob Smith
for Global Game Jam 2015

You need 3 people and 2 gamepads to play this game!

Controls
Player 1 - Keyboard
Player 2,3 - Gamepad

Keyboard Controls
A,S,D,Space,Shift
Numbers- Level Select
Return - Restart
Esc - Exit

Gamepad Controls
360 - Left,Right,LT,RT,RB
PS3 - Left,Right,L2,R2,R1
